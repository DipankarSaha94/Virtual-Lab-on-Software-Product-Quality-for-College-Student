Compatibility

Compatibility is the ability to live and work together without any discrepancy. Compatible product also work in the same setup i.e. Degree to which a product, system or component can exchange information with other products, systems or components, and/or perform its required functions while sharing the same hardware or software environment.

Expectation

Let's consider a website as software product. Now our expectation for this website would be, it is proficient enough to run in different browsers(like chrome, firefox, Internet Explorer) , operating system(like windows, MacOS), mobile devices(like android, IOS).

Example

For example, consider site Google.com, it looks same in all different browser and diferent operating system, so it is compatible. Now let's say you are given a website to build.After the build is complete, you observe that your website looks really good in Chrome but when open form an another browser it changes. The reason is that some of the functions you have written for your website is only limited to some particular browser, then, you can say your website is partially compatible and in quality matric it won't earn you full score.
